* OrgMode
Here are some snippets I made for OrgMode
** Titler
#+begin_src snippet :tangle "/home/haider/.emacs.d/snippets/org-mode/titler"
  # -*- mode: snippet -*-
  # name: Insert Title and Date
  # key: <t
  # --
  ,#+TITLE: ${1:Title}
  ,#+AUTHOR: Haider M
  ,#+DATE: `(let ((x (org-insert-time-stamp nil nil t ))))`
  :PROPERTIES:
  ,#+OPTIONS:
  :END:

  $0
#+end_src
** Source Blocks
You Might want to manually transfer these over (you can see why)
*** Org Source Block
#+begin_src snippet
# -*- mode: snippet -*-
# name: Org Src Block 
# key: <src
# --
# #+BEGIN_SRC $1
$0
# #+END_SRC
#+end_src

*** Org Elisp Source Block
#+begin_src snippet
# -*- mode: snippet -*-
# name: Org Elisp Src Block 
# key: <el
# --
# #+BEGIN_SRC elisp
$0
# #+END_SRC
#+end_src
